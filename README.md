# AES-256-GCM
Advanced Encryption Standard (AES-256) in Galois/Counter Mode (GCM) file encryption / decryption tool written in Python.

# Features
- Recursive folder encryption / decryption
- Temporary file replacement for safe encryption / decryption
- AES-256-GCM authenticated encryption
- Automatic random nonce generation
- Automatic AES-256 key generation
- File size protection (2GB limit which is adjustable)

# Requirements
- cryptography
